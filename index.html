Anatomy of a web page you ask again!

Webpages are not intended to seem similar in any way, form or fashion to one another, nevertheless, they all tend to share some rather similar standard components, like a header and a footer, the main navigation menu used by all visitors to navigate the site, the main section where the content is typically showcased. And, even sidebars are part of the basic anatomy of a website. The following table illustrates the sections of the anatomy of a website, the HTML elements and their syntax along with some very basic information that may help piece things together.

Element Syntax Particularity
header <header> Usually a big strip across the top with a big headline, logo and perhaps a tagline.
-nav <nav> Links to the sites main sections. It's usually represented by menu buttons, links and/ or tabs.
main <main> A big area in the center that contains unique content of a given webpage, for example the video that you want to watch, or the block of news headlines on the site that you like to browse.
-article <article> This element represents even more specific content on a webpage like a news article or a blog.
-section <section> This element represents a generic standalone section of a document, like the different sections of a recipe.
-div <div> The div element is the generic container for flow content It has no effect on the content or layout until styled in some way using CSS.
-aside <aside> Peripheral info, links, quotes, ads etc. Usually contextual to what is contained in the main content.
footer <footer> A strip across the bottom of a page that generally contains fine print, copyright notices, and/ or even contact info.
